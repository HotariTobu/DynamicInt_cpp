# DynamicInt_cpp

`DynamicInt_cpp` is a C++ library that provides a dynamic integer class.

## Description

`DynamicInt` is a dynamic integer class based on a bool array and has no limits.

## Usage

```
auto n = DynamicInt(10);
auto f = DynamicInt(1);
for (auto i = DynamicInt(1); i <= n; ++i) {
  f *= i;
}
```
